## GitHub Pages

To publish this site on GitHub Pages:

1. Create a new repository on GitHub.
2. Add this folder as the repository content.
3. Push the repository to GitHub.
4. In repository settings, enable GitHub Pages from the `main` branch.
5. Use the site URL to access `index.html` and the homework pages.

## Local setup

```bash
git init
git add .
git commit -m "Initial commit for GitHub Pages homework site"
```
